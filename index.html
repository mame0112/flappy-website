About this service

What's flappy?

'flappy' is stress-less communication tool. You will not feel stress any more different from other messaging service.

Stress-less?

One of characteristic feature of 'flappy' is to be disappeared message. If you don't read new message more than 30 minutes, it will disappear automatically and you will not be able to read it anymore.

So what?

This messaging service will not make message receiver reply because you and your friend might missed the message before disappearing. It means message sender can send message without considering situation of receiver. As as result, you can send message more freely than now!
